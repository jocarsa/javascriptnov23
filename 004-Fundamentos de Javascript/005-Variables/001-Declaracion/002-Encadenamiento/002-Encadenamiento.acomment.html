En este código HTML, se ha declarado una variable en JavaScript llamada `edad` y se le ha asignado el valor numérico 45. Luego, se utiliza `document.write` para escribir en el documento HTML el mensaje que incluye el valor de la variable `edad`. Vamos a analizar el código: ```html ``` 1. `var edad = 45;`: Como en el ejemplo anterior, declara una variable llamada `edad` y le asigna el valor numérico 45. 2. `document.write("Tu edad es de: " + edad + " años");`: Utiliza `document.write` para escribir en el documento HTML. El mensaje concatenará la cadena "Tu edad es de: " con el valor de la variable `edad` y la cadena " años". La expresión `"Tu edad es de: " + edad + " años"` es una concatenación de cadenas y el valor de `edad`. En resumen, este bloque de código imprimirá en el documento HTML el mensaje "Tu edad es de: 45 años". Al cargar esta página en un navegador, verás ese mensaje en el cuerpo del documento HTML.
